# Flutter Mini Game

## Overview
An addictive arcade-style mini game built using Flutter, demonstrating the power of Flutter's animation engine and custom painting.

## Key Features
- **Fast-paced Gameplay**: Smooth block-blasting mechanics.
- **Score System**: High score tracking and progression.
- **Sound Effects**: Immersive audio feedback for game actions.
- **Responsive Controls**: Optimized for both touch and keyboard inputs.
- **Vibrant Graphics**: Custom-drawn canvas elements and glowing effects.

## Technical Stack
- **Framework**: Flutter
- **Canvas**: CustomPainter for game logic rendering.
- **Animations**: Flutter AnimationController & Tweens.
- **State Management**: Provider / Simple State.
