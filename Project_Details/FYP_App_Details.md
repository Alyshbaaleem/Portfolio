# Flutter FYP App (University Management)

## Overview
A sophisticated Final Year Project (FYP) application designed to bridge the communication gap between university administration, faculty, and students.

## Key Features
- **Learning Management System (LMS)**: Access course materials, assignments, and grades.
- **Attendance Tracking**: Real-time attendance monitoring for students.
- **Faculty Portal**: Tools for instructors to manage classes and student records.
- **Push Notifications**: Instant alerts for important university announcements.
- **Secure File Sharing**: Upload and download academic documents safely.

## Technical Stack
- **Framework**: Flutter
- **Backend**: Firebase (Auth, Firestore, Cloud Messaging)
- **Architecture**: MVVM / Clean Architecture
- **State Management**: Bloc / GetX
