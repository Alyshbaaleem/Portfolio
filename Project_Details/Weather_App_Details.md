# Weather Flutter App

## Overview
A high-performance weather forecasting application built with Flutter, providing real-time weather updates, detailed forecasts, and interactive UI elements.

## Key Features
- **Real-time Weather**: Current temperature, humidity, wind speed, and weather conditions.
- **7-Day Forecast**: Detailed daily and hourly predictions.
- **Location Detection**: Automatically fetches weather for the current city using GPS.
- **Search Functionality**: Look up weather for any city globally.
- **Dynamic Themes**: UI colors change based on weather conditions (sunny, rainy, cloudy).

## Technical Stack
- **Framework**: Flutter
- **Language**: Dart
- **State Management**: GetX
- **API**: OpenWeatherMap API
- **Local Storage**: GetStorage
