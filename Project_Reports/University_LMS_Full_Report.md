# Full Project Report: University Management LMS 🎓

## 1. Project Overview
University Management LMS ek educational hub hai jo university ki academic activities ko streamline karta hai. Yeh system Software Requirement Specification (SRS) ke mutabiq banaya gaya hai taake students aur faculty ka darmiyan communication behtar ho.

## 2. Core Features & Functionalities
### **A. Academic Portal**
- **Role-Based Access**: Students aur Teachers ke liye alag alag portals aur functionalities.
- **Interactive Dashboard**: Latest announcements, notices, aur alerts ka central view.

### **B. Course & Learning Management**
- **Enrolled Courses**: Student jin courses mein enrolled hai unki list aur syllabus.
- **LMS Resources**: Lectures notes, presentations, aur assignment files download karne ki saholat.
- **Assignment Submission**: Online assignments upload karne ka secure portal.

### **C. Planning & Tracking**
- **Academic Calendar**: Exams, assignment deadlines, aur university holidays ka complete track.
- **Attendance & Grading**: (Based on SRS) Grading system aur attendance record keeping.

## 3. Technical Architecture
- **Framework**: Flutter (Dart)
- **Design Philosophy**: Core emphasis on usability aur accessibility.
- **Architecture**: Structured modular files for calendar, cart (if applicable for resources/fees), category, aur profile management.

## 4. Complete Application Flow
1. **Identity**: Login portal ke zariye Student ya Instructor identity verify karna.
2. **Home Overview**: Dashboard par ja kar naye notices aur alerts check karna.
3. **Learn**: Courses section mein ja kar study material access karna.
4. **Task**: Assignments complete kar ke portal par submit karna.
5. **Plan**: Calendar section mein ja kar upcoming events aur exams ki date check karna.
